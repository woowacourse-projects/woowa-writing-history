# 유연성 강화? 스터디? - 레벨1 글쓰기

```
유연성 강화 스터디 해봤자 얼마나 좋아지겠어? 왜 이런 걸 해야 하지?
```

유연성 강화 스터디라는 이름을 처음 접했을 때, 가장 먼저 든 생각이었다. `유연성 강화`라는 개념이 크게 와닿지 않았고, OT를 들은 후에도 그저 듣기 좋은 말들만 나열하는 것처럼 느껴졌다.

## 어라? 좋은데?

하지만 지금은 너무 좋다. 추상적이라 생각했던 유연성 강화 목표에 가까워졌고, 더 나아졌다고 느끼기 때문이다.

가장 크게 받은 도움은 크루들의 `주기적인 피드백`이다. 만약 혼자이거나, 피드백을 한달 뒤에나 받았다면, 이미 흥미를 잃고 포기했을 것 같다.

더 나은 사람이 되고 싶다는 `막연한 목표보다` 이것만 해도 좋아지려나? 라는 마음으로 내 목표에 가까워질 수 있었다. 유강스는 큰 목표를 세부적인 `실험 계획으로 나누어` 실천하도록 유도하는 방식으로 진행된다. 덕분에 목표 실천에 대한 부담이 줄었고, 능동적으로 실천할 수 있었다.

## 유연성 강화 목표와 실험 계획

```
목표: 함께 문제를 해결하고 싶은 사람이 되고 싶다.

# 실험 계획
1. 주장하기 전 스스로 왜? 3번 묻기
2. 대화를 경쟁이 아닌 협력적인 방향으로 이끌기 위해 적극적인 동의해보기
3. 하루 3번 이상, 바로 반박하고 싶은 순간을 3초 멈춰보기
```

누군가에게 지식을 공유하거나 공유 받는 과정에서 더 견고한 지식을 쌓을 수 있다고 생각한다. 함께 문제를 해결하고 싶은 사람이 되어 그런 환경을 조성하고자 했다.

## 주기적인 피드백

```
주기적인 피드백을 받지 않았다면, 유강스는 평생 숙제로 남았을 것이다.
```

유강스에서 정한 실험 계획은 의식하지 않으면 실천하기 힘든 것들이었다.

혼자서 이를 의식하고 실천하기는 매우 어렵다. 매주 조원들과 피드백을 주고받는 시간을 가진 덕분에 실험 계획을 꽤 오래 의식할 수 있었다. 더 좋았던 점은 가까운 조원들을 실험 대상으로 삼을 수 있었다는 것이고, 매주 내가 잘하고 있는지, 놓친 부분은 없는지 피드백을 받을 수 있었다는 것이다.

매주 피드백을 주고받는 시간이 없었다면, 실험 계획을 세운 것조차 잊어버렸을 것이다.

주기적인 피드백이 가져온 변화는 예상보다 컸다. 이 변화의 범위를 넓혀 일상과 학습에도 적용할 수 있겠다는 생각을 했다. 아침에 크루들과 '오늘은 친절하게 대화하기'를 약속하고 저녁에 서로 피드백을 나눠볼 수 있을 것 같다. 더 나아가 혼자 공부할 때도 적용해볼 수 있을 것 같다.

### 좋았던 피드백

- 소통은 `경청` 즉 잘 듣는 것으로부터 시작인 것 같아.
  - 내 의견을 논리적으로 잘 말하는 것도 중요하지만, 상대방의 의견을 끝까지 잘 들어주고 공감해주는 것, 이후 나의 의견을 표현하는 것이 소통의 한 사이클인 것 같아.
- 해보지 않으면 평생 모르니까 시도해봐.
  - 다 해보고 적정선을 찾아보는 것도 좋을 것 같아
- 대화를 하다가 이전보다 정했던 목표에 가까워졌다는 것을 느꼈어.

## 막연한 목표를 작게 나누자

```
목표가 크면 어디서부터 시작해야 할지 감이 안 잡히고, 실천하기가 어렵다.
```

유강스는 막연하게 느껴지는 유연성 강화 목표를 작은 실험 계획으로 나누어 실천할 수 있도록 도와준다. 크기가 작아지니 실천하기가 훨씬 편했다.

### 실험 계획 수정

실천을 할 수 있다는 것부터 큰 장점이라고 생각한다. 실천을 하다 보면, 실험 계획을 수정하거나, 더 나은 방법을 찾기 쉬운 것 같다. 실제로 실험 계획 2개를 더 구체적으로 수정했다.

```
변경 전: 1. 주장하기 전 스스로 왜? 3번 묻기
변경 후: 1. 질문을 하기 위해 많은 준비를 하지 않고, 그냥 해보기
```

실험해보니 3번 묻는다는 것이 어색하다고 느꼈다. 정확히 3번을 고민해야 하는가? 똑같은 답이 나와도 괜찮은가?

그래서 이전 실험 계획은 2, 3번에 통합하고, 새로운 실험 계획을 추가했다.

서로의 시간은 소중하기 때문에 질문할 때 생각흐름을 모두 전달해야 한다고 생각했다. 그러다 보니 시간이 너무 많이 걸렸고, 가끔은 질문을 포기하게 되는 경우도 있었다. 그래서 질문을 하기 위해 많은 준비를 하지 않고, 그냥 해보는 실험 계획을 세웠다. 해보지 않으면 평생 적정선을 찾지 못할 것 같아서 추가했다.

```
변경 전: 3. 하루 3번 이상, 바로 반박하고 싶은 순간을 3초 멈춰보기
변경 후: 3. 말을 끊고 싶은 순간에 참고 친절하게 호응해보기
```

1번 실험 계획과 비슷한 맥락이다. 하루 3번 참으면 더이상 참지 않을 것인가? 3초만 멈추면 되는 것인가? 그래서 명확한 실험 계획으로 수정했다. 말을 끊고 싶은 순간에 참고 친절하게 호응해보는 실험 계획으로 수정했다.

조원들에게 피드백을 받고 단순히 반박하고 싶은 순간에 참는 것이 아니라, 끝까지 호응하며 상대방의 의견을 들어보기, 즉 경청이 중요하다는 것을 느꼈다. 그래서 실험 계획을 수정했다.

### 실험 계획 수정 후 피드백

- 실험 계획을 수정하면서 갖게 된 마인드셋이 정말 좋은 것 같아!
  - 평생 모르기보다 지금 실천해서 알아보기 굿!
- 실험계획이 적절하지 않다고 생각해서 좀 더 구체적으로 변경하고, 그걸 의식적으로 일상생활에서 적용하려는 모습이 좋은 것 같아.
- 평소에 해보지 않는 걸 해보는게 진짜 좋은 계획인 것 같아.
  - 확실히 장단점을 깨달을 수 있겠다.

작은 실험 계획은 실천하기 쉽게 만들어줬다. 실천이 이루어졌기에 구체적인 피드백을 받을 수 있었다. 계획의 규모가 작아 필요한 수정을 쉽게 적용할 수 있었다. 이 과정이 반복되며 점점 더 효과적인 방법을 발견하는 선순환이 형성된 것 같다.

만약 목표가 크고 막연했다면, 전혀 나아지지 못했을 것 같다.

### 마무리

```
과거의 나처럼 경험하지 않은 사람은 믿기 어려울 정도로 많은 도움을 받았다.
```

앞으로도 선순환(목표를 작게 나누기 + 주기적인 피드백)을 가져가며 더 나은 사람이 되고 싶다. 유연성 강화 스터디를 통해 얻은 인사이트는 앞으로도 계속 활용할 수 있을 것 같다.

단기적으로는 유강스를 유지하고, 장기적으로는 선순환을 모든 분야에 적용해보는 실험을 해보려고 한다.

물론 실험 계획을 세우고 실천하는 것이 불편하고 어색하다. 어렵지만, 더 나아지는 모습이 보여지는게 즐겁다.
