# 우아한테크코스 도서

우아한테크코스 크루들과 코치들의 경험담을 기수별·주제별로 엮어 온라인 도서로 제공합니다.

<Callout type="info">
우아한테크코스를 거쳐간 크루들의 성장 이야기, 코치들의 교육 철학과 시행착오를 한 권의 책처럼 읽을 수 있도록 구성했습니다.
</Callout>

## 기획 의도

우아한테크코스는 2019년 1기를 시작으로 매 기수마다 크루들의 성장 경험과 코치들의 교육 실험이 축적되어 왔습니다. 이 경험담들은 블로그, 회고록, 발표 자료 등 다양한 형태로 흩어져 있습니다.

이를 **기수별로 한 권의 책**으로 엮어, 각 기수의 분위기와 도전, 성장의 맥락을 온전히 전달하고자 합니다.

## 각 기수별 도서의 공통 구성

기수별 도서는 다음과 같은 주제 챕터로 구성됩니다. 기수마다 해당하는 챕터가 다를 수 있습니다.

| 챕터 | 설명 |
|------|------|
| **지원과 선발** | 우테코에 지원한 계기, 프리코스 경험, 합격까지의 여정 |
| **레벨 1 — 프로그래밍 기초** | 페어 프로그래밍, 미션, 코드 리뷰를 통한 기초 체력 다지기 |
| **레벨 2 — 웹 프로그래밍** | 웹 기술 학습, 팀 프로젝트 경험, 기술 스택 확장 |
| **레벨 3 — 팀 프로젝트** | 실전 프로젝트, 협업과 소통, 기술적 의사결정 |
| **레벨 4 — 성장과 취업** | 취업 준비, 레거시 코드 리팩터링, 자기 주도 학습 |
| **글쓰기와 발표** | 글쓰기 미션, 테코톡 발표, 기술 커뮤니케이션 |
| **크루 생활** | 동료와의 관계, 크루 문화, 일상과 추억 |
| **코치의 시선** | 코치들이 본 해당 기수의 이야기, 교육 의도와 시행착오 |
| **수료 후** | 현업에서의 성장, 우테코 경험의 영향, 후배 크루에게 |

## 활용 방법

- **예비 크루**: 기수별 경험담을 통해 우테코의 실제 분위기를 미리 체험
- **현재 크루**: 선배 크루들의 시행착오에서 배우고 자신의 이야기를 기록
- **코치**: 과거 기수의 맥락을 이해하고 교육 설계에 참고
- **수료 크루**: 자신의 기수를 돌아보고 경험을 공유
