# 실패 사례 카탈로그

<Placeholder>7년간의 교육 실험에서 기대한 효과를 얻지 못한 사례들을 정리합니다. 각 사례는 아래 구조를 따릅니다.</Placeholder>

## 사례 구조

각 실패 사례는 다음 구조로 기술합니다:

| 항목 | 설명 |
|------|------|
| **가설** | 어떤 교육적 효과를 기대했는가 |
| **실험** | 구체적으로 무엇을 시도했는가 |
| **결과** | 실제로 어떤 일이 일어났는가 (가능하면 데이터 포함) |
| **분석** | 왜 기대와 달랐는가 |
| **전환** | 이 실패가 어떤 디자인 패턴이나 방침으로 전환되었는가 |

<Callout type="tip">
각 실패 사례의 "전환" 항목은 이 문서의 다른 장(디자인 패턴, 커리큘럼, 평가 등)으로 연결됩니다.
실패가 어떻게 현재의 교육 모델로 진화했는지 추적할 수 있습니다.
</Callout>
