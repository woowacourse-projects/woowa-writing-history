# 디자인 패턴

교육 현장에서 반복적으로 등장하는 문제 상황과 검증된 해결 방법을 패턴 형태로 정리했습니다.
각 패턴은 소프트웨어 디자인 패턴에서 영감을 받아 **문제 → 맥락 → 해결책 → 사례 → 변형** 구조로 기술합니다.

<Callout type="tip">
각 패턴은 독립적으로 참조할 수 있습니다. 교육 현장에서 특정 문제에 직면했을 때 해당 패턴을 찾아 적용해 보세요.
</Callout>
